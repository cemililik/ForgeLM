The full matrix Tüm matris

Features grouped by lifecycle Yaşam döngüsüne göre özellikler

From raw documents through training to deployment artifacts. Every capability is opt-in via YAML and tested in CI. Ham dokümanlardan eğitime, deployment artifact'larına kadar. Her yetenek YAML üzerinden opt-in ve CI'da test ediliyor.

Training & alignmentEğitim ve alignment

Six post-training paradigms, parameter-efficient methods, and distributed configs.Altı post-training paradigması, parametre-verimli yöntemler ve dağıtık konfigürasyonlar.

SFT

Supervised fine-tuning on instruction-formatted JSONL. Multi-dataset mixing with configurable ratios.Talimat formatlı JSONL üzerinde denetimli fine-tuning. Konfigüre edilebilir oranlarla çoklu veri seti karışımı.

DPO & SimPO

Direct Preference Optimization on chosen/rejected pairs; SimPO for reference-free, lower-memory variants.Tercih edilen / reddedilen çiftler üzerinde DPO; referans modelsiz, düşük bellekli varyantlar için SimPO.

KTO

Kahneman-Tversky preference learning on binary feedback. Works when you only have thumbs-up / thumbs-down signals.İkili geri bildirimde Kahneman-Tversky tercih öğrenmesi. Sadece thumbs-up / thumbs-down sinyaliniz varsa çalışır.

ORPO

Single-pass odds-ratio preference optimization. Combines SFT + DPO into one stage; faster wall-clock.Tek geçişli odds-ratio tercih optimizasyonu. SFT + DPO'yu tek aşamaya birleştirir; daha hızlı wall-clock.

GRPO

Group Relative Policy Optimization for reasoning RL. Built-in format/length shaping reward, or your own reward function.Akıl yürütme RL için grup-göreli politika optimizasyonu. Yerleşik format/uzunluk reward'ı veya kendi reward fonksiyonunuz.

QLoRA + DoRA

4-bit NF4 quantization with LoRA. PiSSA, rsLoRA, DoRA variants. Train 7B-13B models on a single 12 GB GPU.LoRA ile 4-bit NF4 kuantizasyon. PiSSA, rsLoRA, DoRA varyantları. Tek 12 GB GPU'da 7B-13B model eğitin.

GaLore

Gradient low-rank projection for full-parameter training without LoRA. Memory at LoRA levels, capacity of full fine-tune.LoRA olmadan full-parameter eğitim için gradient low-rank projeksiyonu. LoRA seviyesinde bellek, full fine-tune kapasitesi.

DeepSpeed & FSDP

Multi-GPU distributed training. ZeRO-2 / ZeRO-3 presets, FSDP, Unsloth backend (2-5× faster).Çoklu-GPU dağıtık eğitim. ZeRO-2 / ZeRO-3 hazır ayarlar, FSDP, Unsloth backend (2-5× hızlı).

Long-context

RoPE scaling, NEFTune noise injection, sliding-window attention, sample packing — train on 32K-128K context.RoPE scaling, NEFTune noise injection, kayar-pencere attention, sample packing — 32K-128K context'te eğitim.

Evaluation & safetyDeğerlendirme ve güvenlik

Benchmarks, judges, harm classification, and the auto-revert that ties them together.Benchmark'lar, judge'lar, zarar sınıflandırması ve hepsini bağlayan otomatik geri alma.

Benchmark integrationBenchmark entegrasyonu

Plug-in lm-evaluation-harness tasks. Per-task floor thresholds; auto-revert if any drops below the floor.lm-evaluation-harness görev entegrasyonu. Görev başı taban eşikleri; biri tabanın altına düşerse otomatik geri alma.

LLM-as-judgeLLM-as-judge

Quality scoring via OpenAI API or a local judge model. Pairwise comparison, single-rubric scoring, ELO-style ranking.OpenAI API veya yerel judge modeliyle kalite skorlaması. İkili karşılaştırma, tek-rubrik skorlaması, ELO sıralaması.

Llama Guard safetyLlama Guard güvenliği

Confidence-weighted scoring across 14 harm categories (S1-S14). Severity tiers, cross-run trend tracking.14 zarar kategorisinde (S1-S14) güven-ağırlıklı skorlama. Ciddiyet katmanları, koşular arası trend izleme.

Auto-revertOtomatik geri alma

If loss, benchmark, or safety thresholds fail, ForgeLM rolls back to the last-good checkpoint and emits a structured incident record.Loss, benchmark veya güvenlik eşikleri başarısız olursa ForgeLM son-iyi checkpoint'e döner ve yapılandırılmış olay kaydı üretir.

Trend trackingTrend izleme

Cross-run safety / quality history. See if a category quietly drifts before it crosses the threshold.Koşular arası güvenlik / kalite geçmişi. Bir kategori eşiği aşmadan önce sessizce kayıyor mu görün.

Exit codesExit kodları

0 / 1 / 2 / 3 / 4 — a public contract for CI/CD. Fail-loud is the default; no silent || true.0 / 1 / 2 / 3 / 4 — CI/CD için kamuya açık kontrat. Sessiz başarısızlık yok; || true yasak.

Data ingestion & auditVeri ingest'i ve denetimi

Raw documents to clean JSONL, with PII / secrets / quality / leakage all surfaced before training.Ham dokümandan temiz JSONL'a — eğitimden önce tüm PII / sırlar / kalite / sızıntı yüzeyde.

Multi-format ingestÇoklu format ingest'i

PDF, DOCX, EPUB, TXT, Markdown → SFT-ready JSONL. Markdown-aware splitter; DOCX tables emit as Markdown tables; PDF header/footer dedup.PDF, DOCX, EPUB, TXT, Markdown → SFT-hazır JSONL. Markdown-bilen splitter; DOCX tabloları Markdown tablo olarak; PDF header/footer dedup.

PII maskingPII maskeleme

Email, phone, credit card (Luhn-validated), IBAN, national IDs (TR / DE / FR / US-SSN). Masks at ingest time, before chunks land in JSONL.E-posta, telefon, kredi kartı (Luhn doğrulamalı), IBAN, ulusal kimlik (TR / DE / FR / US-SSN). Chunk'lar JSONL'a düşmeden ingest sırasında maskelenir.

Secrets scrubbingSırların temizlenmesi

AWS keys, GitHub PATs, Slack / OpenAI / Google tokens, JWTs, full PEM private-key blocks (BEGIN…END), Azure storage strings — redacted with [REDACTED-SECRET].AWS anahtarları, GitHub PAT'leri, Slack / OpenAI / Google token'ları, JWT'ler, tam PEM özel anahtar blokları (BEGIN…END), Azure storage string'leri — [REDACTED-SECRET] ile maskelenir.

Near-duplicate detectionNear-duplicate tespiti

LSH-banded simhash (default, exact recall) or MinHash LSH for >50K-row corpora. O(n × k) typical, streaming, CPU-only.LSH-banded simhash (varsayılan, kesin recall) veya 50K+ satır corpus için MinHash LSH. Tipik O(n × k), streaming, CPU.

Quality filterKalite filtresi

Gopher / C4 / RefinedWeb-style heuristics: low-alpha, abnormal word length, repeated lines, short paragraphs. Conservative — never silently drops.Gopher / C4 / RefinedWeb tarzı heuristik: düşük alfa, anormal kelime uzunluğu, tekrarlayan satırlar, kısa paragraflar. Muhafazakar — sessizce satır düşürmez.

Language detectionDil tespiti

Top-3 language counts per split via langdetect. Useful when your dataset is supposed to be Turkish but 12% slipped in as English.langdetect ile split başına top-3 dil sayısı. Veri setiniz Türkçe olmalıyken %12'si İngilizce kaymışsa fark eder.

Cross-split leakageSplit-arası sızıntı

Catches the most expensive evaluation bug: train rows that also appear in validation/test. Audit refuses to certify a leaky split.En pahalı değerlendirme hatasını yakalar: validation/test'te de görünen train satırları. Audit sızdıran split'i onaylamaz.

Annex IV provenanceAnnex IV provenance

Audit report is automatically embedded in the EU AI Act Article 10 governance artifact when present at training time.Eğitim sırasında mevcutsa audit raporu EU AI Act Article 10 governance artifact'ına otomatik gömülür.

Enterprise & MLOpsKurumsal ve MLOps

Built for pipelines: containers, webhooks, structured outputs, air-gap, model cards.Hatlar için yapıldı: konteynerler, webhook'lar, yapılandırılmış çıktı, air-gap, model card.

Webhook notificationsWebhook bildirimleri

Slack and Teams alerts on training start, success, failure, auto-revert. Structured payloads — pipe to your incident tool.Eğitim başlangıcında, başarıda, hatada, otomatik geri almada Slack ve Teams alert'leri. Yapılandırılmış payload — incident aracınıza bağlayın.

Experiment trackingDeney takibi

W&B, MLflow, TensorBoard via report_to. Pluggable, not hardcoded. Use what your team already uses.report_to üzerinden W&B, MLflow, TensorBoard. Pluggable, hardcoded değil. Ekibinizin zaten kullandığını kullanın.

JSON output modeJSON çıktı modu

--output-format json emits a machine-readable envelope — perfect for piping to jq, GitHub Actions, or your custom dashboard.--output-format json makine-okunur bir zarf üretir — jq'a, GitHub Actions'a veya özel dashboard'unuza pipe için ideal.

Air-gap operationAir-gap operasyonu

No mandatory internet calls. Pre-cache models / datasets / tokenizers; run training in an isolated network with full feature parity.Zorunlu internet çağrısı yok. Model / veri seti / tokenizer'ı önceden cache'leyin; izole ağda tam özellik paritesiyle koşun.

Docker & composeDocker ve compose

Official Dockerfile and docker-compose.yml. Reproducible runs across local, on-prem, and cloud — same image, same behaviour.Resmi Dockerfile ve docker-compose.yml. Yerel, on-prem ve cloud arasında yeniden üretilebilir koşular — aynı imaj, aynı davranış.

Model card auto-genOtomatik Model Card

HuggingFace-compatible README with metrics, benchmarks, training config snapshot. Drop-in for HF Hub publish.Metrikleri, benchmark'ları ve training config snapshot'ını içeren HuggingFace uyumlu README. HF Hub yayını için drop-in.

GPU cost estimationGPU maliyet tahmini

Auto-detection across 18 GPU profiles (A100 / H100 / RTX 4090 / L40S / etc.) with per-run cost tracking against your hourly rate.18 GPU profili (A100 / H100 / RTX 4090 / L40S / vb.) otomatik algılama, saatlik tarifenize göre koşu başı maliyet izleme.

Synthetic data pipelineSentetik veri hattı

Teacher-to-student distillation with --generate-data. Augment your dataset; cite the teacher in the model card automatically.--generate-data ile öğretmenden öğrenciye damıtma. Veri setinizi büyütün; öğretmen otomatik olarak model card'a referans olur.

Model mergingModel birleştirme

TIES, DARE, SLERP, linear merge of multiple adapters via --merge. Compose specialists into a generalist.--merge ile birden çok adapter için TIES, DARE, SLERP, lineer birleştirme. Uzmanları genel bir modelde birleştirin.
