Other channels Diğer kanallar

  • GitHub Issues

    Reproducible bug? File an issue with config + log excerpt — the team triages within 48 hours. Tekrarlanabilir hata mı? Config + log alıntısıyla bir issue açın — ekip 48 saat içinde triage eder.

    Open an issue → Issue aç →

    Open-ended questions, design ideas, or "is this the right tool for X" — discuss with the community. Açık uçlu sorular, tasarım fikirleri veya "X için doğru araç mı" — toplulukla tartışın.

  • Security disclosure Güvenlik açıklaması

    Found a vulnerability? Use the form on the left with topic Security disclosure. We acknowledge within 72 hours and respect responsible-disclosure timelines. Bir güvenlik açığı mı buldunuz? Soldaki formu Security disclosure konusuyla kullanın. 72 saat içinde teyit ederiz ve sorumlu-açıklama takvimine saygı duyarız.
